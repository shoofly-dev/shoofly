What Shoofly Blocks

🎯 Blocks Prompt Injection

Stops malicious instructions hidden in web content, emails, and documents from hijacking your agent. Catches 8 injection patterns including jailbreaks, instruction overrides, and base64 payloads.

🔐 Blocks Credential Theft

Intercepts attempts to read or exfiltrate your API keys, GitHub tokens, and AWS credentials before they leave your machine.

🚫 Blocks Unauthorized Writes

Stops writes to sensitive paths — /etc/, ~/.ssh/, LaunchAgents — before your system is modified.

🔁 Stops Runaway Agents

Detects agents stuck in loops: repeated tool calls, call floods, read-write cycles, URL hammering.

🔍 Blocks Malware in Tool Results

Catches malicious instructions embedded in web fetches, API responses, and file reads — the attack vector Anthropic calls their #1 risk.

🔑 Finds Secrets Before Attackers Do

One-shot scan of your project directory for 10 credential patterns. CI-integrable.
