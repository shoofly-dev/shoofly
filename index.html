What We Catch

🎯 Prompt injection detection

Catches 8 injection patterns including instruction suppression, jailbreak keywords, and base64 payloads.

🔐 Credential sniffing alerts

Including Anthropic API keys, GitHub tokens, and AWS credentials.

🚫 Out-of-scope write detection

Flags writes to sensitive paths — /etc/, ~/.ssh/, LaunchAgents — before damage is done.

🔁 Runaway loop detection

Detects 4 loop patterns: repeated tool calls, call floods, read-write cycles, URL hammering.

🔍 Tool response injection

Catches malicious instructions embedded in tool call results, HTML comments, JSON keys, and API responses.

🔑 Secret detection at rest

One-shot scan of your project directory for 10 credential patterns. CI-integrable.

Basic catches threats after the fact. Advanced stops them before they reach your agent.

  • Two-layer prevention — daemon monitors + alerts, hook blocks before execution
